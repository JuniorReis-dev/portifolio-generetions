RideBro

O RideBro é um aplicativo de gerenciamento de caronas desenvolvido em equipe, proporcionando uma maneira eficiente de organizar e acompanhar viagens compartilhadas. Criado com Java, Spring Boot, MySQL e JPA, sua API REST permite cadastro de usuários, veículos e viagens, garantindo uma experiência segura e estruturada. A aplicação segue uma arquitetura modular, facilitando futuras expansões como autenticação, reservas e avaliações de usuários.
